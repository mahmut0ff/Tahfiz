# Система управления выпускниками TAHFIZ

## Описание
Модуль для управления выпускниками медресе TAHFIZ. Позволяет переводить активных студентов в статус выпускников, отслеживать их достижения после окончания обучения и вести контакты.

## Основные функции

### 1. Управление выпускниками
- **Перевод студентов в выпускники** - автоматическое изменение статуса студента
- **Просмотр списка выпускников** с фильтрацией по году и группе
- **Детальная информация** о каждом выпускнике
- **Редактирование данных** выпускников
- **Удаление записей** с возвратом статуса "активный студент"

### 2. Достижения выпускников
- **Добавление достижений** после окончания обучения
- **Категоризация достижений** (образование, карьера, религиозная деятельность и др.)
- **Хронология достижений** с датами и описаниями
- **Удаление достижений**

### 3. Контактная информация
- **Актуальные контакты** выпускников
- **Текущая деятельность** выпускников
- **Примечания** и дополнительная информация

## Структура модуля

### Модели
- `Graduate` - основная информация о выпускнике
- `GraduateAchievement` - достижения выпускника после окончания

### Представления (Views)
- `graduate_list` - список выпускников с фильтрацией
- `graduate_detail` - детальная информация о выпускнике
- `graduate_create` - создание записи о выпускнике
- `graduate_update` - редактирование информации
- `graduate_delete` - удаление записи
- `make_graduate` - перевод студента в выпускники
- `achievement_create` - добавление достижения
- `achievement_delete` - удаление достижения

### Формы
- `GraduateForm` - основная форма для создания/редактирования выпускника
- `MakeGraduateForm` - специализированная форма для перевода студента
- `GraduateAchievementForm` - форма для добавления достижений

## Использование

### Перевод студента в выпускники
1. Перейти на страницу деталей студента
2. Нажать кнопку "Перевести в выпускники" (доступна только для активных студентов)
3. Заполнить форму с информацией о выпуске
4. Подтвердить перевод

### Управление достижениями
1. Открыть страницу выпускника
2. Нажать "Добавить достижение"
3. Заполнить информацию о достижении
4. Выбрать категорию и дату

### Фильтрация выпускников
- По имени или номеру диплома
- По году выпуска
- По группе выпуска

## Права доступа
- **Администраторы** - полный доступ ко всем функциям
- **Операторы** - полный доступ ко всем функциям
- **Преподаватели** - только просмотр
- **Студенты** - доступ запрещен

## Интеграция
Модуль интегрирован с:
- Системой студентов (изменение статуса)
- Системой групп (связь с группой выпуска)
- Системой пользователей (права доступа)

## Статусы студентов
- `active` - Активный студент
- `graduated` - Выпускник
- `inactive` - Неактивный студент
- `expelled` - Исключенный студент

## URL-маршруты
- `/graduate/` - список выпускников
- `/graduate/<id>/` - детали выпускника
- `/graduate/create/` - создание выпускника
- `/graduate/<id>/update/` - редактирование
- `/graduate/<id>/delete/` - удаление
- `/graduate/make-graduate/<student_id>/` - перевод студента
- `/graduate/<id>/achievement/create/` - добавление достижения
- `/graduate/achievement/<id>/delete/` - удаление достижения
